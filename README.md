Этот проект представляет собой информационный веб-портал, разработанный в рамках учебного курса для Aitu College. Основная цель приложения — создать удобную платформу для отслеживания спортивных событий, новостей и расписания внутри учебного заведения.

В основе стека лежит связка React для интерфейса и Node.js с Express для серверной логики. Данные хранятся в нереляционной базе MongoDB, что позволяет гибко управлять контентом. В проекте реализована базовая структура клиент-серверного взаимодействия, а фронтенд уже адаптирован для деплоя через GitHub Pages.

Технические особенности:

Компонентный подход на React для быстрой работы UI.

Централизованное управление данными через API на бэкенде.

Версионный контроль и работа с Git для прозрачности разработки.

Проект задуман как готовое решение для автоматизации информационных потоков в колледже, с возможностью дальнейшего масштабирования и добавления личных кабинетов пользователей. Для запуска достаточно склонировать репозиторий, установить зависимости через npm и настроить подключение к базе данных.
